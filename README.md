# San-Valentin
Flores Rojas animacion
Animacion artistica romantica en html y puro css, diseñada para regalos digitales, landing pages especialesdemostraciones creativas y proyectos emocionales de alto impacto visual.

Esta animación representa un campo de flores animadas con efectos de iluminación, estrellas fugaces y un mensaje central personalizable, ideal para San Valentín, aniversarios o regalos personalizados.

 Características principales

 Sin librerías externas (HTML + CSS puro)

 Animación fluida con @keyframes, transform, gradients y blur

 Efectos visuales avanzados (estrellas fugaces, iluminación sutil, profundidad)

 Mensaje romántico completamente personalizable

 Carga rápida y compatible con navegadores modernos

 Diseño responsive (se adapta a distintas pantallas)

 Personalización

Puedes personalizar fácilmente el mensaje principal editando el siguiente bloque en index.html:

<h1>Feliz San Valentín</h1>

Ejemplo:

<h1>Para María </h1>

También puedes modificar colores, velocidad de animaciones y cantidad de flores desde el archivo CSS.
